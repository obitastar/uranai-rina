# 四柱推命 鑑定データ集

御福まつかわや 店頭診断アプリ用の四柱推命データ。Claude Code でこのデータを参照して Web アプリを構築します。

## 構成

データは性質の違う2層に分かれています。

### 第1層：計算用の確定データ（`core.ts`）
答えが一意に定まるマスターデータと判定ロジック。

- `TENKAN` … 天干10種（甲乙丙丁戊己庚辛壬癸）。五行・陰陽つき
- `CHISHI` … 地支12種（子丑寅卯…）。五行・陰陽・蔵干・時刻つき
- `KANSHI_60` … 60干支の循環表
- `SOSEI` / `SOKOKU` … 五行の相生・相剋
- `getTsuhensei(日主, 対象天干)` … 通変星を算出する関数
- `getJuniunsei(日主の漢字, 地支index)` … 十二運星を算出する関数

> **注意：命式（年柱・月柱・日柱・時柱）の算出には、別途 `lunar-javascript` 等の暦ライブラリを推奨。**
> 節入り時刻（立春での年の切り替わり等）は天文計算が絡むため、自前実装は避け、ライブラリ出力をこのデータと突き合わせて鑑定する設計が安全です。

### 第2層：鑑定文データ
すべて一般的な四柱推命の体系に基づき、御福まつかわやのトーンでオリジナルに書き下ろしたもの（特定の書籍・サイトからの転載なし）。

- `nisshu.ts` … 日主10種：本質・恋愛・仕事・キーワード・象徴
- `tsuhensei.ts` … 通変星10種：意味・才能・注意点
- `interpretation.ts`
  - `JUNIUNSEI` … 十二運星12種：エネルギー・メッセージ
  - `GOGYO_BALANCE` … 五行バランス5種（多い時/少ない時）
  - `NENUN` … 年運5グループ：今年の運勢

## 使い方（鑑定の組み立て）

1. 暦ライブラリで生年月日時から命式（4本の柱の天干地支）を算出
2. 日柱の天干 = **日主**。`NISSHU[日主]` で本質・恋愛・仕事を取得
3. 各柱の天干に `getTsuhensei` を適用 → `TSUHENSEI[星]` で才能傾向
4. 各柱の地支に `getJuniunsei` を適用 → `JUNIUNSEI[星]` でエネルギー
5. 命式全体の五行を集計 → `GOGYO_BALANCE` で偏りを読む
6. 今年の干支から年運グループを判定 → `NENUN` で今年の運勢

## 店頭体験での見せ方（推奨）

専門的にしすぎず、お客さんが楽しめる範囲に絞る：
**日主（本質）→ 五行バランス → 恋愛 → 仕事 → 今年の運**

## 検証

`npx tsx validate.ts` で全データの網羅性と、計算・鑑定文の連結を確認できます。

## データを増やしたい時（Claude Code への指示例）

> 「`nisshu.ts` と同じ構造で、日主と通変星の組み合わせごとの相性文を追加して。文体は既存データに合わせ、四柱推命の一般的解釈に基づきオリジナルで書き下ろすこと」

このように既存ファイルを「お手本」として渡すと、トーンと構造を保ったまま拡張できます。
